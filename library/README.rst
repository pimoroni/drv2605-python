Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install drv2605``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/drv2605-python``
-  ``cd drv2605-python``
-  ``sudo ./install.sh``
